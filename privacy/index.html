LBTimer Help Contact

  • LBTimer requests the user's permission to access its own data in the user's Google Drive™. The permission is requested using Google's OAuth2 authentication system. The data is stored in a hidden folder accessible only to LBTimer app. The data is accessed using Google's Drive API. You can read more about this hidden folder here.

    For the e-mail notification feature, as well as for the functionality of directly sending an e-mail to the developer from the extension panel, LBTimer requests the user's permission to send an e-mail in their name from their Gmail account. E-mails are sent using Google's Gmail API. If those features are not used, the permission is not even requested.

    You can review Google's privacy policies here.

    This website features a Twitter button. You can view Twitter's privacy policy here.
